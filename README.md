# Gosh

> **🏗️ The project is still in a very early stage. | 该项目还处于非常早期阶段。**

Gosh is a powerful shell with LLM enhancement.

Actually it is not a shell, but an enhanced wrapper for shells like zsh / bash. You can use your familiar shell as usual, but it is better than before.

---

Gosh 是一个 LLM 增强的 shell 。

实际上它不是一个 shell ，而是 zsh / bash 等 shell 的增强包装。你可以像往常一样使用你熟悉的 shell ，但是它比过去更好了。
