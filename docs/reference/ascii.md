# ASCII

| 编码     | 部分作用                   | 输入方式                   |
|--------|------------------------|------------------------|
| `\x01` | 光标移到行首                 | `Ctrl` + `A`           |
| `\x02` | 光标向左移动                 | `Ctrl` + `B`           |
| `\x03` | 中断当前正在运行的命令或进程         | `Ctrl` + `C`           |
| `\x04` | 删除光标所在处的字符、关闭输入（ EOF ） | `Ctrl` + `D`           |
| `\x05` | 光标移到行尾                 | `Ctrl` + `E`           |
| `\x06` | 光标向右移动                 | `Ctrl` + `F`           |
| `\x07` |                        | `Ctrl` + `G`           |
| `\x08` | 删除光标前一个字符              | `Ctrl` + `H`           |
| `\x09` | 制表符                    | `Ctrl` + `I`           |
| `\x0a` | 创建新行，相当于回车             | `Ctrl` + `J`           |
| `\x0b` | 剪切从光标到行尾的所有字符          | `Ctrl` + `K`           |
| `\x0c` | 清空屏幕                   | `Ctrl` + `L`           |
| `\x0d` | 回车                     | `Ctrl` + `M`           |
| `\x0e` | 下一个指令，相当于下键            | `Ctrl` + `N`           |
| `\x0f` | 光标跳到新行，相当于回车           | `Ctrl` + `O`           |
| `\x10` | 上一个指令，相当于上键            | `Ctrl` + `P`           |
| `\x11` | 恢复屏幕输出                 | `Ctrl` + `Q`           |
| `\x12` | 搜索历史命令                 | `Ctrl` + `R`           |
| `\x13` | 暂停屏幕输出                 | `Ctrl` + `S`           |
| `\x14` | 将光标处的字符与前一个字符互换位置      | `Ctrl` + `T`           |
| `\x15` | 剪切从光标到行首的所有字符          | `Ctrl` + `U`           |
| `\x16` | 转义控制字符                 | `Ctrl` + `V`           |
| `\x17` | 剪切光标前的一个单词             | `Ctrl` + `W`           |
| `\x18` | 光标在行首和当前位置之间切换         | `Ctrl` + `X`           |
| `\x19` | 将剪切的内容粘贴到光标处           | `Ctrl` + `Y`           |
| `\x1a` | 暂停当前正在运行的前台任务          | `Ctrl` + `Z`           |
