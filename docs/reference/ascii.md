# ASCII

| 编码          | 名称  | 部分作用                   | 输入方式                   |
|-------------|-----|------------------------|------------------------|
| `\x00` `\0` | NUL | 空字符                    |                        |
| `\x01`      | SOH | 光标移到行首                 | `Ctrl` + `A`           |
| `\x02`      | STX | 光标向左移动                 | `Ctrl` + `B`           |
| `\x03`      | ETX | 中断当前正在运行的命令或进程         | `Ctrl` + `C`           |
| `\x04`      | EOT | 删除光标所在处的字符、关闭输入（ EOF ） | `Ctrl` + `D`           |
| `\x05`      | ENQ | 光标移到行尾                 | `Ctrl` + `E`           |
| `\x06`      | ACK | 光标向右移动                 | `Ctrl` + `F`           |
| `\x07` `\a` | BEL |                        | `Ctrl` + `G`           |
| `\x08` `\b` | BS  | 删除光标前一个字符              | `Ctrl` + `H`           |
| `\x09` `\t` | HT  | 制表符                    | `Tab` / `Ctrl` + `I`   |
| `\x0a` `\n` | LF  | 创建新行，相当于回车             | `Ctrl` + `J`           |
| `\x0b` `\v` | VT  | 剪切从光标到行尾的所有字符          | `Ctrl` + `K`           |
| `\x0c` `\f` | FF  | 清空屏幕                   | `Ctrl` + `L`           |
| `\x0d` `\r` | CR  | 回车                     | `Enter` / `Ctrl` + `M` |
| `\x0e`      | SO  | 下一个指令，相当于下键            | `Ctrl` + `N`           |
| `\x0f`      | SI  | 光标跳到新行，相当于回车           | `Ctrl` + `O`           |
| `\x10`      | DLE | 上一个指令，相当于上键            | `Ctrl` + `P`           |
| `\x11`      | DC1 | 恢复屏幕输出                 | `Ctrl` + `Q`           |
| `\x12`      | DC2 | 搜索历史命令                 | `Ctrl` + `R`           |
| `\x13`      | DC3 | 暂停屏幕输出                 | `Ctrl` + `S`           |
| `\x14`      | DC4 | 将光标处的字符与前一个字符互换位置      | `Ctrl` + `T`           |
| `\x15`      | NAK | 剪切从光标到行首的所有字符          | `Ctrl` + `U`           |
| `\x16`      | SYN | 转义控制字符                 | `Ctrl` + `V`           |
| `\x17`      | ETB | 剪切光标前的一个单词             | `Ctrl` + `W`           |
| `\x18`      | CAN | 光标在行首和当前位置之间切换         | `Ctrl` + `X`           |
| `\x19`      | EM  | 将剪切的内容粘贴到光标处           | `Ctrl` + `Y`           |
| `\x1a`      | SUB | 暂停当前正在运行的前台任务          | `Ctrl` + `Z`           |
